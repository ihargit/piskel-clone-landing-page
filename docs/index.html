General description

The Piskel clone is simplified clone of the https://www.piskelapp.com/, a web-based tool for Spriting and Pixel art.

Piskel clone DEMO: https://ihargit.github.io/piskel-clone/

Functional

  1. Canvas sizes: 32x32 / 64x64 / 128x128 units
  2. Tools
    • Pen
    • Color select
    • Paint bucket
    • Eraser
  3. Preview
    • Small animation preview window in the top right corner
    • Full screen mode
  4. Frames
    • Ability to add/delete a frame
    • Ability to reorder a frame via Drag and Drop
    • Ability to duplicate a frame
